      Requirements
Use a 2D array of characters to store the picture.
Use:
* for drawing pixels
_ for empty spaces
Implement functions to draw the following shapes:
Rectangle
Line
Triangle
Circle
The editor should support:
Adding objects to the picture
Deleting objects from the picture
Modifying existing objects in the picture
Write a function to display the picture on the screen.
Store information about each shape so that it can be modified or deleted later. The stored information should include:
Shape type
Coordinates
Size/Dimensions
Other necessary parameters

Whenever an object is:

Added
Deleted
Modified

The program must redraw the entire picture.

Use structures to maintain a list of objects in the picture.
